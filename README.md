# 京ふれーず ～洛中リアル・京ことば辞典～

和風モダンでポップな京ことば辞典アプリです。

- 京ことばを「直訳」と「意訳」で表示
- 129語を7ジャンルに分類
- カードごとにカラーイラスト付き
- カード単位とサイト全体の共有ボタン付き
- 静的HTMLなので `index.html` を開くだけで動作

## Files

- `index.html`: アプリ本体
- `assets/illustrations/`: カード用イラスト
- `assets/icons/`: ファビコン／アプリアイコン

## GitHub Pages

GitHub Pagesで公開する場合は、リポジトリの Settings から Pages を開き、Source を `Deploy from a branch`、Branch を `main / root` に設定してください。
